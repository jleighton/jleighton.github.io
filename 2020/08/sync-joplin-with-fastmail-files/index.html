Sync Joplin with Fastmail Files

Date Sun 30 August 2020 Category howto

How to setup [Joplin](1) Synchronisation using Fastmail Files (WebDAV)

Synchronisation target: WebDAV

Webdav URL: [myfiles.fastmail.com](https://myfiles.fastmail.com/){directory in fastmail files you want to save your notes in}

WebDAV User: fastmail username

WebDAV Password: [Application Specific Password](2) for Files
